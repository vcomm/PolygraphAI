# PolygraphAI
Analysis of the truth of the user's answer to the question asked.

**Gemini API Key (required)**

- **Why:** This app requires a Gemini API key for requests to the Gemini model.
- **Obtain a free key (brief):**
	1. Sign in or create a Google Cloud account and open the Google Cloud Console.
	2. Create a new project (or select an existing one).
	3. Enable the Vertex AI / Gemini API from "APIs & Services" (search for "Vertex AI" or "Gemini").
	4. Go to "APIs & Services" → "Credentials" → "Create credentials" → "API key" and copy the generated key.
	- Google may offer free trial credits or a free tier; check billing/quotas in the console.

- **Store the key locally :**
That the Gemini API Key must be entered only once when first logging into the application in UI and stored in localStorage.
